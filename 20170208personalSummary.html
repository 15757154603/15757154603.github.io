jQuery DOM 操作

  • jQuery可以获得元素的内容和属性。
  • jQuery中非常重要的部分，就是操作DOM的能力。
  • DOM=Document Object Model。（文档对象模型）

jQuery中获得和设置内容的方法

  • text()：获取或设置元素的文本内容。
    html()：获取或设置元素的xml（包括文本内容）。
    val()：获取或者是元素的val值。
    val(property，string)：获取或者是元素的属性值。

jQuery DOM

  • W3C DOM 和 HTML DOM 是国际上通用的DOM，简称为DOM。
  • Microsoft DOM是IE浏览器适用的，还有一些其他的DOM。

课外小知识

  • inpu元素中，placeholder属性是输入框中的提示信息，当placeholder属性和value属性一同存在时，value中的属性值会被覆盖。
  • BOM=Browser Object Model（浏览器对象模型）。

jQuery中添加/删除元素

  • append()/preappend()在被选元素下的前部或结尾添加元素。
    prependTo()/appendTo()把被选元素追加给某个元素，效果是一样的，但是括号中元素的位置相反。
  • after()/before()在被选元素之前或之后添加元素。
    insertAfter()/insertBefore()把被选元素追加给某个元素，效果是一样的，但是括号中元素的位置相反。
  • remove()：删除被选元素以及所有子元素。empty()：删除被选元素中的所有子元素。

jQuery中添加/删除类

  • addClass()：向元素添加一个或多个类；
    removeClass()：从被选元素中删除一个或多个类；
    toggleClass()：对被选元素进行类的切换。
