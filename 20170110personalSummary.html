js

  • 是动态的、基于原型的脚本语言。
  • 值类型有五种：分别是数字、布尔、字符串、Null、Undefine。引用类型：对象、数组、函数。
  • JS的渲染是自上而下，从左到右。

函数

  • 常用函数声明:
    function fun(){}
  • 表达式:
    var fun = function(){}
  • 匿名声明(也是立即执行函数):
    (function(){})()
  • 动态声明:
    var fun = new function("a=1","b=1","return a+b")
  • arguments是函数当中的伪数组。
  • 函数的参数可以是函数名，同时一个函数也可以在另一个函数内部调用。
  • 函数是JS的一等公民，函数式声明会被提前

事件调用

  • 常用方法在标签内部直接调用:
    div id="" onclick="fun()"
  • 外部调用:
    div.addEventlistener("click",fun,true)
  • 另一种:
    var div = document.getElementById("box");
    div.onclick=fun

DOM和BOM

  • DOM一般是指HTML，DOM也是W3C里面的Dom Document Object Model。
  • BOM一般是指浏览器对象，它的核心对象的Windows。

闭包

  • 访问一个函数内部声明的一个函数，例子如下:
    function fun(){
    var num = 1;
    return num ;
    }
    console.log(fun())

全局变量和局部变量

  • this指向：1、指向标签 2、指向windows 3、指向对象。
  • "this"指向的是调用时，所处的环境对象。

权限提升

  • 例：
    var fun = function(){
    console.log("111");
    }
    
				function fun(){
    console.log("222");
    }
				
    fun();会出现111.

apply和call

  • 可以改变函数中this的指向，使用方式为fun.apply(obj,[i])
