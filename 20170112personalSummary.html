内置对象

  • 分为三种：分别是Array、Date、Error。
  • arr.push()指在数组末尾增加数据；arr.unshift()指在数组第一个增加数据。
  • str.splice("a")指在str字符串中以a元素分开两个或者多个字符串；str.contact()连接字符串。

小知识

  • for(){}数组循环 ；for in 非数组循环；sort 进行数组的升序排列。
  • 数组的升序排列：
    arr.sort(function(a,b){
    return a-b;
    })
  • 数组的降序排列：
    arr.sort(function(a,b){
    return b-a;
    })

Error

  • 浏览器内置的报错对象。
  • err.message，可以获取Error的报错信息。
  • ReferenceError引用错误。
  • RangeError引用错误。
  • TypeError类型错误。
  • SyntaxError语法错误。

小知识

  • try是捕获错误，catch是处理错误
    try{
    }catch(){
    }
  • clientX、clientY时间被触发时鼠标指针的位置。
  • obj.preventDefault;取消浏览器默认行为

事件

  • W3C推荐写法：document.addEventlistener("要触发的事件，去掉on"，函数或者函数名在外部引用(注意去掉括号)，false)
  • 日常推荐写法：document.on+事件名=function(){}

事件流

  • 分为：事件冒泡和事件捕获
