之前的学习总结

  • 网页布局：分为静态布局、自适应布局、流式布局、响应式布局、弹性布局。
  • 流式布局：大小该表，位置不变；自适应布局：大小不变。位置改变。弹性布局，通常用display：flex。

弹性盒子布局

  • display：flex 有浮动效果；flex-grow：扩展；flex-shrink：收缩；
  • flex-flow：有两个属性：row横向排布；column纵向排布；flex-direction向下排列；align-items：flex-start左上角；flex-end左下角。

cmd命令

  • mkdir：创建文件夹；write：写字板；mspaint：画图板；mstsc：远程连接；notepad：记事本；syncapp：创建公文包；shrpubw：创建共享文件夹； rmdir：删除文件夹；cd：进入；move：移动；copy：复制；type：打开。

数据库命令

  • insert into 表单名 values 增加
  • delete from 表单名 where 删
  • update 表单名 set where 改
  • select 查

WEB存储

  • 分为document.cookie、local.storage、session.storage.
  • document.cookie的优点是兼容性好。缺点是不安全、存储量太小。
  • local.storage是永久存储，存储大小约5M左右，存储到本地客户端。
  • session.storage是临时存储，网页刷新、关闭后就会消失。
  • session.storage、local.storage。都属于window对象下的。而cookie属于document。session.storage、local.storage的区别是大小不一，安全不一，但是方法相同。

UI框架中的Bootstrap

  • 它的优点：1、兼容性好；2、拥有新特性；3、适合项目开发。
  • .container 类用于固定宽度并支持响应式布局的容器。.container-fluid 类用于 100% 宽度，占据全部视口（viewport）的容器。
  • .col-xs- 超小屏幕 手机 (<768px);.col-sm- 小屏幕 平板 (≥768px);.col-md- 中等屏幕 桌面显示器 (≥992px) ; .col-lg- 大屏幕 大桌面显示器 (≥1200px)
  • .col-md-offset-* 类可以将列向右侧偏移。对于被删除的文本使用 del> 标签。对于没用的文本使用 s> 标签。
