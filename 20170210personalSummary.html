jQuery 效果动画

  • 它包括显示隐藏、滑动、淡入淡出、和自定义动画。

显示隐藏

  • show();hide();toggle()显示与隐藏进行切换。

滑动

  • slideDown()向下滑动;slideUp()向上滑动;slideToggle();向上滑动与向下滑动的切换。

淡入淡出

  • fadeIn();fadeOut();fadeToggle()淡入淡出进行切换;fadeTo()透明度进行切换。;

自定义动画 animate()

  • animate({params},speed,callback);params 参数定义形成动画的 CSS 属性； speed 参数规定效果的时长。它可以取以下值："slow"、"fast" 或毫秒。 callback 参数是动画完成后所执行的函数名称。
  • jQuery animate() - 使用队列功能.逐一运行这些 animate 调用。
